WELCOME TO CHIP-IO
{{ version }}

A user-friendly, up-to-date web application for transcription factor gene regulatory analysis with enhancer and tissue-specific features.

AM I IN THE RIGHT PLACE?

  1. Are you working with transcription factors?

    Perfect. ChIP-IO currently handles only transcription factor oriented data sources.

  2. Are you interested in adult humans?

    Good. ChIP-IO is currently limited to adult humans and is standardized to the GRCh38 reference genome.

  3. Are you looking for information on transcription factor regulatory target genes?

    Excellent. This is the primary goal of ChIP-IO.

  4. Would you like this regulatory information to be tissue-specific and generally customizable?

    Hopefully! Transcription factor activity is context dependent. ChIP-IO was designed to handle these unmet needs.

GETTING STARTED

Select the "CONSTRUCT QUERY" tab to perform a custom set of TF-gene regulatory analysis and download the bundled output.

If you would prefer to directly download selected analysis presets, or are simply interested in directly downloading ChIP-Seq/other datasets on file, instead navigate to the "DOWNLOADS" tab above.

Navigate to the "ABOUT" tab above for more information on the technical aspects of the site.
